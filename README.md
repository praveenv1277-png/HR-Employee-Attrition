# HR-Employee-Attrition


##  Project Overview

This project focuses on analyzing HR Attrition data to understand employee turnover patterns and key influencing factors. The objective is to generate meaningful insights that help organizations improve employee retention and workforce planning using data visualization.


##  Data Source

The dataset is an HR Attrition dataset containing employee-related information such as age, gender, department, job role, salary, experience, and attrition status. It provides a detailed view of workforce behavior and attrition trends.
 Source Description: ChatGPT
 Domain: Human Resource Analytics (HR Analytics)


##  Tools & Technologies

* Microsoft Excel – Data Source
* Power BI – Dashboard & Visualization
* Power Query – Data Cleaning & Transformation
* DAX – Calculated Columns & Measures


##  Data Cleaning & Preparation

* Removed duplicate records to ensure data accuracy
* Handled missing/null values appropriately
* Standardized categorical fields (Gender, Department, Attrition)
* Converted data types for consistency
* Created calculated columns like Attrition Count & Attrition Rate
* Organized data for efficient analysis and reporting


##  Exploratory Data Analysis (EDA)

* Analyzed overall attrition rate across the organization
* Compared attrition across departments and job roles
* Studied the impact of age and experience on attrition
* Used slicers for dynamic filtering (department, gender)
* Identified trends using bar charts, pie charts, and line visuals


##  Key Insights

* High attrition is concentrated in specific departments.
  This indicates possible issues like workload imbalance or limited growth opportunities.

* Employees with less experience show higher attrition rates.
  This highlights the need for better onboarding and early career support.

* Younger employees tend to leave more frequently.
  They are more likely to switch jobs for better opportunities.

* Salary has a strong influence on employee retention.
  Lower salary groups show higher attrition levels.

* Certain job roles consistently experience higher attrition.
  This may be due to repetitive tasks or low job satisfaction.

* Attrition varies across different time periods.
  This helps identify peak exit periods for employees.

* Gender has minimal impact on attrition trends.
  Job-related factors play a more significant role.


##  Recommendations

* Focus on high-attrition departments with targeted retention strategies
* Improve compensation and employee benefits
* Provide clear career growth paths for employees
* Enhance employee engagement and satisfaction programs
* Offer training and development opportunities
* Continuously monitor attrition trends using dashboards


##  Types of Analysis

### 1. Descriptive Analysis

Summarizes employee data to understand overall attrition trends and workforce distribution.
Helps identify which departments or groups have the highest attrition.

### 2. Diagnostic Analysis

Analyzes factors like salary, job role, and experience to find reasons behind attrition.
Helps understand why employees are leaving the organization.

### 3. Predictive Analysis

Uses historical patterns to estimate future attrition trends.
Helps identify employees who are at risk of leaving.

### 4. Prescriptive Analysis

Provides actionable recommendations to reduce attrition.
Supports decision-making to improve employee retention and satisfaction.


##  Conclusion

This project provides a clear understanding of employee attrition and its key influencing factors.
The analysis shows that attrition is mainly affected by salary, experience, and job role.
Departments with higher attrition require focused attention and improvement strategies.
Data visualization helps in identifying patterns and making better decisions.
Overall, the project supports organizations in improving employee retention using data-driven insights.
